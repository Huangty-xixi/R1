📋 逐状态详解

🧭 [1] nav_turn_open — 导航到开放目标 + 转向90°

项目内容
入口AppZone1_Start()，导航到 (open_target_x_m, open_target_y_m) + Yaw 90°
红方→FIELD_RIGHT(-90°)，蓝方→FIELD_LEFT(90°)
每帧peek导航 + 等 YawHeadingCtrl_ParallelLegSettled() + nav_leg_done() 同时满足
✅ 成功→ ② reverse_slow_to_limit
❌ 失败odom失效 / BAD_CONFIG → abort

🔄 [2] reverse_slow_to_limit — 后退靠边检测

项目内容
入口从①或⑩到达后
每帧VY = -40.0 后退，检测堵转：|VY|≥2.0 + 堵转轮数≥3 + 持续180ms
✅ 成功堵转触发 → ③ shift_right_monitor
❌ 失败超时6s → abort

🔍 [3] shift_right_monitor — 扫矿监控

项目内容
每帧 run_grab_monitor() 核心逻辑：
· 夹爪 idle→wait_close_delay 检测 → latch → 进clamp_wait
· 视觉Y坐标在 [y_lo, y_hi] 区间: 累计stable_ticks，允许自动夹取
· 超出上/下界: flip扫方向 + pull_back减速
· 速度控制: margin减速 × anchor减速(min 0.6)
· Y区带滞回(6区: below/lo_band/mid/hi_band/above + unknown)
✅ 成功夹爪 latch → ④ clamp_wait

🔧 [4] shift_right_clamp_wait — 等夹爪完成

项目内容
每帧 轮询 ClampHeadCtrl_GetState():
· dock_ok → exit_success
· upright_hold + 物体存在 + 持续2s → exit_success
· idle + 物体不在 + 持续200ms → 重试(最多8次)或FLOW_THROUGH
✅ 成功V5变更 → clamp_wait_exit_success() → 一律进入⑤ advance_turn180
V4: skill_lap≠0时直接done → V5: 统一走advance_turn180
🔁 重试物体丢失 + 重试<8 → ③ shift_right
❌ 失败重试≥8 → FLOW_THROUGH? → exit_success : abort

↻180° [5] advance_turn180 — 转180°返回

项目内容
每帧PostCommand(turn_180) + 导航到 (return_target_x_m, return_target_y_m)
等 YawHeadingCtrl_ParallelLegSettled() + nav_leg_done()
✅ 成功一律 → ⑥ wait_r1_release
❌ 失败yaw180失败 / odom错误 → abort

📡 [6] wait_r1_release — 等松爪信号 V5 V6超时终点 V9:CC/AA分流

项目内容
入口从⑤完成 / V6全局超时强制跳入
每帧清除运动override → 检查 r1_pending≠0 → 退出
第一圈: CC..DD帧触发 第二圈: AA..BB帧触发
r1_pending每帧 poll_r1_release_sig():
lap0 → CC..DD 帧 (R1按按钮)
lap1 → AA..BB 帧 (R1发任务)
V5守卫防重复 · V9: lap0用CC帧,免AA依赖
V9: CC/AA双帧分流 · #if APP_MATCH_SKILL_Z12门控(竞技赛仅AA) | V7: z1_mission_handled=0 移至此入口(advance_turn180→wait_r1_release)，不再在Lap切换时解锁
出口 wait_r1_exit(): z1_mission_handled=1, R1Link_ClearNewMission(), NotifyDockOk()
第一圈(skill_lap==0): skill_lap=1, ClampHeadCtrl_Init() → ⑩ nav_turn_lap2 (z1_mission_handled保持=1，阻挡残余帧)
单圈/第二圈(skill_lap≠0): → ⑦ post_wait_rotate → ⑧ done

🔄 [7] post_wait_rotate — 延时+抬升+旋转 V6 V10

项目内容
入口从 wait_r1_exit 单圈/第二圈完成 (skill_lap≠0)
延迟post_wait_rotate_delay_ms (默认5000ms)
V10 进入即 main_lift_position = main_lift_p4 抬升主轴
旋转方向红方(APP_ZONE2_RED_SIDE=1): YawHeadingCtrl_RunFieldDir(RIGHT)
蓝方(APP_ZONE2_RED_SIDE=0): YawHeadingCtrl_RunFieldDir(LEFT)
✅ 成功YawHeadingCtrl_IsBusy()==0 → ⑧ done
❌ 失败—
