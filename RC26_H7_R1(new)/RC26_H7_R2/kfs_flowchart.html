④ 上下伸缩 ★ 四模式（CH5 循环切换）★ 重点
★ CH5 LOW 边沿 → 四模式循环：0→1→2→3→0（每次按一次切换）
模式 = 速0 | 速1 伸缩速度直驱 | 位2/3 位置环 PID 四档 | above/below 共用 kfs_flex_pos_param

> 速度模式（0/1）

模式0 — below 速度（伸缩下放）
公式: below_cmd = (CH2 - 992) × 200
仅遥控模式有效；全自动无速度直驱
模式1 — above 速度（收放）
公式: above_cmd = (992 - CH2) × 200
CH2 方向与 below 相反；above/below 互斥输出

> 位置模式（2/3） ★ NEW

模式2 — below 位置环 | above=0 | 反馈: kfs_below.round_cnt | 输出: +raw×200 | 切入时记录 base_rounds
模式3 — above 位置环 | below=0 | 反馈: kfs_above.round_cnt | 输出: -raw×200 | 切入时记录 base_rounds
① 切入位置模式
base_rounds = 当前电机.round_cnt
target_pos = pos0
积分/微分清零
CH2 边沿（位置模式）
CH2 ≥1500 → pos+1（上限 pos3）
CH2 ≤500 → pos-1（下限 pos0）
② 目标圈数 (pos_rounds 偏移)
target_rounds = base_rounds + pos_rounds[pos_idx]

③ 位置环 PID（above/below 共用 kfs_flex_pos_param） ★ NEW

error = target_rounds - round_cnt
integral += error   [clamp: ±pos_i_limit]
derivative = error - last_error
output = P×error + I×integral + D×derivative
[clamp: ±max_speed]
④ 缩放 + 方向
below: speed_cmd = output × 200
above: speed_cmd = -output × 200
⑤ 速度环
电机.PID_Calculate(speed_cmd)
档位pos_rounds[i]目标圈数 (round_cnt)
pos0pos_rounds[0] ← 切入时当前位置base_rounds + pos_rounds[0]
pos1pos_rounds[1]base_rounds + pos_rounds[1]
pos2pos_rounds[2]base_rounds + pos_rounds[2]
pos3pos_rounds[3]base_rounds + pos_rounds[3]
★ 全自动模式 (full_auto_control)： 类似 main_lift_position，auto 代码直接设 cmd 枚举
用法： kfs_below_cmd = kfs_below_cmd_p2; → below 位置模式，目标 pos2
kfs_below_cmd = kfs_below_cmd_stop; → 停

模式切换时重置 base_rounds + 档位 PID 状态，防 mode 变化时跳变 ★ NEW
非遥控/非全自动： kfs_above & kfs_below 速度指令置 0，位置环状态复位

场景代码效果
below 到 pos2kfs_below_cmd = kfs_below_cmd_p2;自动切 below 位置模式，目标 pos2，记录 base_rounds，位置环跟踪
