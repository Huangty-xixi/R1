▶ 详细时间线
① idle
初始化：复位 + 设置机构位置
清激光激增标志 · 首轮 (get_kfs_round==0): three_kfs→p1
主轴 main_lift: HIGH_TO_LOW→p0 | GROUND_HIGHEST→p4 | 其他→p3
吸盘: 按当前three_kfs位置开启对应吸盘 (sucker1+2 / sucker1+3 / sucker1+4 / sucker2+3+4)
② spin_front_to_p2 · 1200ms
前臂转到 p2 位置
🔃 kfs_spin_position = kfs_spin_p2
纯等待，不动机器人底盘。前臂从 p1 旋转到 p2 方向。
③ chassis_forward · 2000ms
底盘前进取件
🚗 vy = +20.0 (前进) · kfs_spin = p2
底盘以 vy=20 前进 1.3 秒，前臂保持在 p2 方向。
⚠ 激光防掉桩：检测到测距激增立刻清VY停车，继续后续动作。
完成后：清除 VY override · 前臂回 kfs_spin→p1
主轴抬升: GROUND_HIGHEST→p4 | 其他→p3
标记 chassis_fwd_done = 1 (允许 Zone2 提前返回)
④ spin_front_to_p1 · 1200ms
前臂回 p1 + 关闭吸盘 1
🔃 kfs_spin = p1 · sucker1_state = 0
前臂旋转回 p1，同时关闭吸盘1释放KFS。
⑤ wait_after_close_s1 · 2000ms
等待吸盘释放稳定
🔃 kfs_spin→p2 · main_lift 保持高位
等待 2.5 秒让吸盘完全释放、KFS 稳定。完成后前臂再转到 p2。
⑥ main_lift_to_p1 · 1200ms
主轴保持在抬升位
⬆ main_lift 保持 p4 或 p3
每帧持续设 main_lift（HIGH_TO_LOW 除外），等待 1.2 秒稳定。
⑦ wait_front_p2_done · 200ms
三个KFS旋转到下一个位置
🔃 three_kfs: p1→p2 / p2→p3 / p3→p4
三个KFS整体转一格，为下次取件准备。等待 200ms 完成。
⑧ done
完成清理
ClearChassisOverride · flow_mode=flow_none
get_kfs_round = 1 (后续轮次不再强制 three_kfs→p1)
▶ 参数: g_process_get_kfs_tune
参数值说明
spin_front_to_p2_ms1200前臂从p1转到p2等待时间

spin_front_to_p1_ms1200前臂从p2回p1 + 关吸盘等待
wait_after_close_s1_ms2500吸盘释放后稳定等待
