⚠️ PutKFS 起始状态可变 — 取决于取了多少个 KFS

GetKFS 不一定取满3发。取几个，弹夹就转几格，PutKFS 就从当前位置反转退弹。
取了几个 KFSGetKFS后 three_kfs 位置哪些位置有弹PutKFS 轮数PutKFS 起始 action
1 个 P2 P1 1 轮 three_kfs: P2→P1，关吸盘1
2 个 P3 P1, P2 2 轮 three_kfs: P3→P2，关吸盘2 → P2→P1，关吸盘1
3 个 P4 P1, P2, P3 3 轮 three_kfs: P4→P3→P2→P1，依次关吸盘3→2→1
💡 核心逻辑：PutKFS 不需要知道取了多少个。只要看当前 three_kfs_position 是否 > P1（弹夹未回到起点）， 就反转一格、退一发，直到回到 P1 为止。跟 GetKFS 完全对称。

不需要计数器，只看 three_kfs_position：while (pos > P1) { 反转一格; 关对应吸盘; kfs_above 推弹; }
